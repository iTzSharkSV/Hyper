# Hyper

A CLI to bootstrap new projects!

Generate an Advanced Project Template in seconds.

## Installing

```bash
npm install -g @sharksv/hyper
# or
yarn global add @sharksv/hyper
```

## Usage/Examples

```bash
$ hyper <commands> [options]
```

```bash
# Show help
hyper help

# Initilize a new project
hyper init
```
